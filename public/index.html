🌐 HTTP代理服务

标准HTTP代理服务 - 支持环境变量和curl -x参数

使用示例：

环境变量方式: export https_proxy="https://your-site.netlify.app"

curl -x参数: curl -x https://your-site.netlify.app https://api.github.com/users/octocat

自定义请求头: curl -H "x-request-line: GET https://httpbin.org/get HTTP/1.1" https://your-site.netlify.app

🚀 快速开始

1. 设置环境变量

export https_proxy="https://your-site.netlify.app"
export http_proxy="https://your-site.netlify.app"

2. 正常使用curl

curl "https://api.github.com/users/octocat"
curl "https://httpbin.org/get"

3. 取消代理设置

unset https_proxy http_proxy

🔧 测试命令

直接测试

curl -x https://your-site.netlify.app https://httpbin.org/get

curl -H "x-request-line: GET https://api.github.com/users/octocat HTTP/1.1" \
     https://your-site.netlify.app

📋 支持的协议

  • ✅ HTTP 和 HTTPS
  • ✅ GET, POST, PUT, DELETE 等所有HTTP方法
  • ✅ 请求头和请求体转发
  • ✅ 文件下载和上传

⚠️ 注意事项

  • 请合理使用代理服务，遵守相关法律法规
  • 某些HTTPS网站可能不支持代理
  • 建议添加访问控制和安全措施
