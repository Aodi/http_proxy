🎬 TheMovieDB 反向代理

这是一个基于 Netlify Functions 的 TheMovieDB API 反向代理服务。

使用方法

将你的请求发送到代理端点，所有路径、参数、请求头和请求体都会自动转发到 api.themoviedb.org。

GET https://your-site.netlify.app/.netlify/functions/proxy/3/movie/550?api_key=YOUR_API_KEY
POST https://your-site.netlify.app/.netlify/functions/proxy/3/movie Authorization: Bearer YOUR_ACCESS_TOKEN Content-Type: application/json {"title": "Test Movie"}

支持的请求

  • ✅ 所有 HTTP 方法 (GET, POST, PUT, DELETE, PATCH)
  • ✅ 查询参数
  • ✅ 请求头
  • ✅ 请求体
  • ✅ CORS 支持

⚠️ 注意事项

  • 需要有效的 TheMovieDB API 密钥或访问令牌
  • 遵循 TheMovieDB API 的使用条款和速率限制
  • 代理仅用于开发测试，生产环境请直接使用官方 API
